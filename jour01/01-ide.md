# Environnement de développement

- aller vous connecter sur votre compte gmail 
- ouvrir un nouvel onglet
- rechercher "Google Colab"
- cliquer sur le bouton "Nouveau notebook"

# fichier .ipynb / Jupiter Notebook

- prendre des notes de cours via des zones de markdown
- vous pouvez aussi écrire du Python et l'exécuter 
- 
- toutes les librairies scientifiques sont déjà installées
- `numpy` => tableau NDarray (Tableau à N Dimensions)
- `matplotlib` => dessiner des graphiques
- `pandas` => Excel version python
- `scikit learn` => ML (machine Learning)
- `pytorch` => Deep Learning (réseau de neurone artificiel)

# possibilité de choisir le hardware qui va exécuter votre code

- par défaut : CPU
- mais vous pouvez changer et choisir un GPU T4 => Nvidia Tesla (installer cuda framework Nvidia qui permet de faire communiquer votre code en Python avec une carte graphique)
